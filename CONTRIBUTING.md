# Contributing

Thanks for your interest in BPB Lite.

## Scope

This repo is the public BPB Lite surface, not the private production Based Pings system.

Please keep contributions aligned with:

- public token triage
- compact scan outputs
- attribution clarity
- testable confidence logic
- safe plugin-style packaging

## Development

```bash
npm test
node scripts/scan-token.js <solana-contract-address>
```

## Pull requests

Good PRs usually include:

- a clear user-facing improvement
- tests when logic changes
- concise docs updates when behavior changes
- no private thresholds, secrets, or production bot internals

## Ground rules

- prefer evidence over hype
- prefer honest unknowns over forced certainty
- keep the repo usable for public demos and hackathon review
